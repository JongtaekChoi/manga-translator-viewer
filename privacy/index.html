개인정보 처리방침

최종 업데이트: 2026년 4월 15일

만화 번역 뷰어(이하 "본 확장")는 이용자의 프라이버시를 최우선으로 고려하여 설계되었습니다. 본 확장은 자체 서버를 운영하지 않으며, 이용자의 어떠한 개인정보도 수집·저장·전송하지 않습니다.

1. 수집하는 정보

본 확장은 이용자로부터 어떠한 개인정보도 수집하지 않습니다.

  • 개인 식별 정보(이름, 이메일, 연락처 등)를 요청하거나 수집하지 않습니다.
  • 이용자의 열람 기록, 방문 페이지, 검색어 등 활동 데이터를 추적하지 않습니다.
  • 분석 도구(Google Analytics 등) 또는 제3자 추적 스크립트를 사용하지 않습니다.

2. 이용자가 직접 제공하는 정보의 저장

아래 정보는 이용자가 확장을 사용하기 위해 직접 입력하는 것으로, 이용자의 브라우저 로컬 저장소(chrome.storage.local / localStorage)에만 저장됩니다.

  • OpenAI API 키 — 번역 요청을 위해 필요합니다. 이용자 본인이 OpenAI로부터 발급받은 키로, 본 확장 개발자에게 전송되지 않습니다.
  • 작품 컨텍스트 — 이용자가 입력한 작품명·용어·말투 등 번역 참고 정보.
  • 모델 선택 값 — 사용하려는 OpenAI 모델명(예: gpt-4o-mini).

이 정보들은 이용자의 브라우저를 벗어나지 않습니다. 확장을 제거하거나 브라우저 데이터를 삭제하면 모두 제거됩니다.

3. 제3자 서비스와의 데이터 전송

본 확장은 만화 이미지 번역 기능을 제공하기 위해 OpenAI API(https://api.openai.com)에 직접 요청을 보냅니다. 전송되는 데이터는 다음과 같습니다.

  • 이용자가 "번역" 버튼을 클릭한 만화 이미지 (base64 인코딩)
  • 이용자가 입력한 작품 컨텍스트(있는 경우)
  • 이용자의 OpenAI API 키 (인증 헤더)

이 데이터는 OpenAI의 개인정보 처리방침 및 API 데이터 이용 정책에 따라 처리됩니다. 본 확장 개발자는 OpenAI에 전송된 데이터에 접근할 수 없습니다.

4. 본 확장이 요구하는 권한

  • activeTab — 이용자가 확장 아이콘을 클릭했을 때에만 현재 탭에 접근합니다.
  • storage — 이용자의 설정(API 키 등)을 브라우저 로컬에 저장합니다.
  • host_permissions: <all_urls> — 만화 이미지를 가져와 번역하기 위해 필요합니다. 어떤 데이터도 외부로 전송하지 않습니다.
  • host_permissions: api.openai.com — OpenAI API를 호출하기 위해 필요합니다.

5. 데이터 보안

모든 민감 정보는 이용자의 브라우저 로컬에만 저장되며, 본 확장 개발자의 서버로 전송되지 않습니다. OpenAI로의 통신은 HTTPS로 암호화됩니다. 이용자는 언제든 브라우저 설정에서 확장의 저장 데이터를 삭제할 수 있습니다.

6. 어린이 개인정보 보호

본 확장은 만 14세 미만의 어린이로부터 개인정보를 의도적으로 수집하지 않습니다.
