주요 기능

🌐

어느 사이트에서나

만화가 표시되는 모든 페이지에서 이미지 위에 "번역" 버튼이 자동으로 나타납니다. 사이트별 설정 불필요.

⚡

한 번의 API 호출

OpenAI Vision이 말풍선 검출·OCR·번역을 한 번에 처리합니다. 작품 컨텍스트를 입력하면 캐릭터 말투까지 살립니다.

🔒

프라이버시 우선

API 키는 브라우저 로컬 저장소에만 보관됩니다. 중간 서버 없이 OpenAI로만 직접 전송됩니다.

설치 방법

1

ZIP 파일 다운로드

상단의 "Chrome 확장 다운로드" 버튼을 눌러 압축 파일을 받고 해제합니다.

2

Chrome 확장 페이지 열기

주소창에 chrome://extensions 입력 후, 오른쪽 위의 "개발자 모드"를 켭니다.

3

확장 프로그램 로드

"압축해제된 확장 프로그램을 로드합니다" 클릭 → 해제한 폴더 안의 extension 폴더를 선택합니다.

4

OpenAI API 키 설정

확장 아이콘을 클릭해 팝업을 열고, OpenAI에서 발급받은 키를 입력합니다. 키는 로컬에만 저장됩니다.

5

만화 페이지에서 사용

일본어 만화 페이지에 접속하면 각 이미지 위에 "번역" 버튼이 나타납니다. 클릭하면 말풍선 위치에 한국어가 오버레이됩니다.
